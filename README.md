# NewsHub - Modern News Website

A clean, responsive news website built with HTML and CSS.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Multiple Categories**: World, Technology, Business, and Sports news sections
- **Featured Articles**: Hero section highlighting breaking news
- **Article Pages**: Detailed article view with related content
- **Trending Section**: Shows most popular articles
- **Modern UI**: Clean design with smooth transitions and hover effects

## Structure

```
NewsHub/
├── index.html              # Homepage
├── pages/
│   ├── world.html         # World news category
│   ├── technology.html    # Technology news category
│   ├── business.html      # Business news category
│   ├── sports.html        # Sports news category
│   └── article.html       # Article detail page
├── assets/
│   └── css/
│       ├── style.css      # Main stylesheet
│       └── article.css    # Article page styles
└── README.md

```

## How to Use

1. Open `index.html` in your web browser to view the homepage
2. Navigate through different categories using the navigation menu
3. Click on any article card to view the full article
4. The site uses placeholder images - replace them with actual images as needed

## Customization

- **Colors**: Edit the color scheme in `style.css` (primary color is #ff6b6b)
- **Images**: Replace placeholder images with actual news images
- **Content**: Update article titles, descriptions, and content in HTML files
- **Categories**: Add or modify news categories by creating new HTML pages

## Browser Support

Works on all modern browsers including Chrome, Firefox, Safari, and Edge.
